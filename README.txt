# exiaSaver

Avant d'utiliser tout cela, il faut que vous recompilez chaque fichier en .c !

Pour cela, voici les lignes à rentrer dans votre console :

    gcc launcher.c -o launcher

    gcc termSaver1.c -o termSaver -lncurses

    gcc termSaver2.c -o termSaver

    gcc stats.c -o stats
    
    
Vous êtes fin pret à utiliser le programme ! Tapez :

    ./launcher
    
Amusez-vous bien :p
